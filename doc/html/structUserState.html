Detailed Description

User State Information passed between DMA and application-specific drivers while changing configuration, and reading state/statistics.

  • LinkState can be LINK_UP or LINK_DOWN
  • When DMA driver sets the test state in the user driver, TestMode can be a combination of TEST_STOP, TEST_START and ENABLE_LOOPBACK.
  • When DMA driver gets the test state from the user driver, TestMode can be a combination of TEST_IN_PROGRESS and ENABLE_LOOPBACK.
